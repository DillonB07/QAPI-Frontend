About QAPI

QAPI is an API that I've worked on as a hobby project. It currently has {{questions}} questions with more being added regularly!

Dizzle

Dizzle is a website that gives you a random quiz question for you to answer. It had its own API, but I decided to move it into a seperate project so that anyone can use it.

Statistics

A statistics page for the amount of questions in different topics and the types of questions.

Interactive Testing

An interactive page that lets you test the endpoints and shows your what information needs to be passed.

GitHub

The source code for QAPI is available as a GitHub repository!

Endpoints

/random-question

Returns a random question from a list of {{questions}} questions!

/topic/{topic}/random-question

Returns a random question for the given topic

/type/{type}/random-question

Returns a random question of a given type

/statistics/total

Returns the amount of questions

/statistics/type/{question_type}

Returns the amount of a certain type of question

/statistics/topic/{topic}

Returns the amount of questions in a certain topic
